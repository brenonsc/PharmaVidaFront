# PharmaVidaFront :hospital::pill: 

Este repositório contém o código fonte do frontend da farmácia PharmaVida. O projeto foi desenvolvido em React e inclui as seguintes funcionalidades:

- Uma página Home com informações sobre a farmácia
- Componentes Navbar e Footer para navegação e conteúdo de rodapé
- Funcionalidades CRUD para categorias

<br>

## Testando o software :man_scientist:

Para rodar o projeto, siga as seguintes instruções:

1. Instale as dependências do projeto com o seguinte comando:

```
yarn install
```

2. Inicie o servidor de desenvolvimento com o seguinte comando:

```
yarn dev
```

3. O projeto será aberto no navegador na porta 5173. Para acessar a página Home, acesse `http://localhost:5173`.

<br>

## **Funcionalidades** :abc:

A página Home do projeto apresenta informações básicas sobre a farmácia, como nome, endereço, telefone e horário de funcionamento. O componente Navbar fornece links para as principais páginas do site, enquanto o componente Footer contém informações de contato e direitos autorais.

As funcionalidades CRUD para categorias permitem que os usuários criem, editem e excluam categorias de produtos. Para acessar essas funcionalidades, acesse [Categorias](http://localhost:5173/categorias) ou [Cadastrar Categoria](http://localhost:5173/cadastrarcategoria) na NavBar.
